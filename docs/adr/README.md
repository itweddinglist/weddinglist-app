# Architecture Decision Records

Acest folder conține deciziile arhitecturale ale proiectului weddinglist-app.

## Ce e un ADR?

Un ADR documentează o decizie arhitecturală importantă: contextul, opțiunile evaluate,
decizia luată și consecințele ei. Scopul e să înțelegi de ce s-a ajuns la o soluție,
nu doar ce e soluția.
